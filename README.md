# Search Application

A simple search application built with HTML, CSS, and JavaScript. The application allows users to search through data and filter results based on views.

## Features

- Search by title
- Filter results by view count range
- Real-time search suggestions
- Responsive grid layout

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the JSON server:
```bash
npm run server
```

3. Open `index.html` in a browser

## Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON Server for backend API
